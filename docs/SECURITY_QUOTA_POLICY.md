# FieldPress AI Security + Quota Policy

## Owner/Internal
Owner account:
- role: owner
- plan: enterprise_internal
- is_internal: true

Owner is not blocked by quota but usage is still logged.

## Free
- 3 AI generate per day
- 3 minutes audio per day
- demo only

## Basic
- 100 AI generate per month
- 180 minutes audio per month
- top up allowed

## Pro
- 500 AI generate per month
- 1200 minutes audio per month
- CMS, SEO, collaboration

## Security Guards
- API rate limit
- AI rate limit
- Upload rate limit
- Audio MIME validation
- Max upload size 25MB
- Security event logging
- RLS on usage and security logs
